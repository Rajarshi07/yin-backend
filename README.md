# YIN Backend

### Technologies:
- Django
- SQLlite

### Requirements
- Python
- Pip

### How to Run Locally
```bash
# clone the git repo
git clone https://github.com/sandippakhanna/yin-backend.git

# install all packages
cd yin-backend
pip install -r requirements.txt

# run the server
cd youth_news
python manage.py runserver
```
Now open http://localhost:8000/ in your browser.
